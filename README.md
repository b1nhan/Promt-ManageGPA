# Promt-ManageGPA
Prompt này được thiết kế để chatbot hỗ trợ tính điểm môn học, GPA học kỳ và GPA tổng.

> Lưu ý: Đây là prompt mẫu – nên dùng để tham khảo cấu trúc hoặc tùy chỉnh lại cho phù hợp với từng hệ thống hoặc trường học cụ thể.

- Dấu [] là placeholder, thay bằng số hoặc dữ liệu thực tế phù hợp.

- Prompt hiện hoạt động tốt nhất với Gemini, chưa được kiểm thử trên các model khác.

- Có thể bổ sung thêm ngữ cảnh hoặc quy tắc tính điểm riêng theo từng trường để tăng độ chính xác.
